Netgen is hiring

User interface / front-end developer

Netgen traži user interface / front-end web developera koji će nam se pridružiti u zagrebačkom uredu.

Nudimo rad na međunarodnim projektima u iskusnom timu, kontinuirano učenje i usavršavanje te poticajnu i ugodnu radnu okolinu.

Kao tvrtka, fokusirani smo na razvoj kompleksnih web stranica i web aplikacija. Usluge koje pružamo primarno koriste tvrtke i digitalne agencije u Europi, vezane uz eZ Publish CMS platformu. O Netgenu više može saznati na našim web stranicama te društvenim profilima.

Tvoj tipičan radni dan u Netgenu uključivat će riješavanje sljedećih zadataka:

  • Izrada HTML/CSS markupa, samostalno ili u timu s ostalim front-end developerima
  • Implementacija javascript komponenti za web sučelja
  • Implementacija web aplikacija, zasnovanih na Javascript frameworcima
  • Proaktivno sudjelovanje u optimizaciji Netgenovog procesa izrade web sučelja
  • Dijeljenje znanja s ostalim članovima tima
  • Produciranje dobro organiziranog, dokumentiranog i testiranog programskog koda
  • Surađivanje s back-end developerima koji kreiraju REST sučelja, twig predloške i podatkovne stukture koje je potrebno integrirati u web sučelje

Znanja i vještine koje tražimo i cijenimo:

  • Iskustvo u korištenju suvremenih front-end tehnologija, optimizacijskih i best-practices tehnika u razvoju web sučelja
  • Iskustvo u izradi responzivnih web sučelja za desktop, tablet i mobilne uređaje
  • Sposobnost samostalne organizacije i zadržavanja produktivnosti u situacijama kada je potrebno raditi na više projekata istovremeno
  • Razumijevanje procesa razvoja web stranica i aplikacija
  • Osnovno iskustvo s naprednim JavaScript frameworcima (Backbone, Angular, Ember i sl.)
  • Osnovno iskustvo rada sa sustavima za verzioniranje koda (Git)
  • Osnovno iskustvo u radu sa shellom na Linux/Unix operacijskim sustavima
  • Preciznost pri izradi i testiranju web sučelja
  • Sposobnost samostalnog rada i rada u timu
  • Dobre komunikacijske vještine, sposobnost dobre suradnje s kreativnim timom i back-end developerima
  • Vrlo dobro poznavanje engleskog jezika u govoru i pismu

Dodatno iskustvo koje je dobro došlo:

  • Iskustvo na projektu gdje se web sučelje razvijalo iterativnim pristupom u suradnji s klijentom i ostalim članovima projektnog tima
  • Sudjelovanje u razvoju nekog open source proizvoda
  • Iskustvo u komunikaciji s klijentom
  • Iskustvo u radu s API sučeljima
  • Razumijevanje UX disciplina i načela, poput usability testinga, UI dizajna, informacijske arhitekture, i sl.
  • Sposobnost izrade wireframeova i prototipa

Još nekoliko važnih detalja :)

Ako smatraš da trenutno ne možeš u potpunosti zadovoljiti gornje kriterije, a svejedno misliš da bi za tebe Netgen bio pravi izbor svakako se javi.

Ured se nalazi na lako dostupnoj lokaciji u blizini Jarunskog jezera, idealnog za rekreaciju prije ili na kraju radnog dana. Svi Netgenovci sudjeluju na ljetnom kampu, međunarodnoj konferenciji koju organiziramo jednom godišnje u rujnu (https://www.facebook.com/eZSummerCamp). Trudimo se osigurati sve što je potrebno za dodatno stručno usavršavanje, od sudjelovanja na konferencijama do potrebne stručne literature i ostalih edukacijskih materijala.

Tvoja prijava bi trebala sadržavati:

  • kratak životopis
  • popis relevantnih projekata
  • GitHub/Bitbucket profil ili sl. (opcionalno)
Prijavu možeš poslati na careers@netgenlabs.com.
